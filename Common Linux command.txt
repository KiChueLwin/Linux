Navigating and working with directories

.ls - list files and directories
.find - find files in directory tree
.pwd - get present working directory 
.mkdir - make directory 
.cd - change directory 
.rmdir - remove directory 


Printing file and string contents

.cat - print file contents
.more - print file contents page by page
.head - print first N lines of files
.tail - print last N lines of files
.echo - print string or variable value 


Networking 
.hostname - print hostname
.ping - send packets to URL and print response
.ifconfig - display or configure system network interfaces 
.curl - display contents of file at URL 
.wget - download file from URL
